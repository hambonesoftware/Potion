# Agent: RepoAgent

## Purpose
Create protocol-backed repositories with async methods; wire SwiftData implementation.

## Tasks
- `PlantsRepo`, `VillagesRepo`, `ActivitiesRepo`, `DocsRepo`, `LogsRepo` protocols.
- Default implementations using SwiftData.
- Simple CRUD SwiftUI screens to verify flow.

## Tests
- Repo unit tests using an in-memory container.
